# Client Framework for Minecraft

## Install

### Maven

```xml
<repositories>
    <repository>
        <name>Enaium</name>
        <id>Enaium</id>
        <url>https://enaium.gitee.io/maven/</url>
    </repository>
</repositories>

<dependencies>
    <dependency>
        <groupId>cn.enaium</groupId>
        <artifactId>cf4m</artifactId>
        <version>1.1.0</version>
    </dependency>
</dependencies>
```

### Gradle

```groovy
dependencies {
    compile group: 'cn.enaium', name: 'cf4m', version: '1.1.0'
}

repositories {
	maven {url 'https://enaium.gitee.io/maven/'}
}
```
