# 安装

# Maven

<repositories>
    <repository>
        <name>Enaium</name>
        <id>Enaium</id>
        <url>https://enaium.gitee.io/maven/</url>
    </repository>
</repositories>

<dependencies>
    <dependency>
        <groupId>cn.enaium</groupId>
        <artifactId>cf4m</artifactId>
        <version>1.0.0</version>
    </dependency>
</dependencies>

# 使用

注意

当前仅有Event、Module和Setting

# Instance

创建主类.

public enum Example {

    instance;

    public CF4M cf4M = new CF4M(this);

}

# Start and Stop

在游戏启动和停止时使用Start和Stop.

Example.instance.cf4M.start(); Example.instance.cf4M.start();

# Event

提示

CF4M内置了2个Event(KeyboardEvent,UpdateEvent)

注意

你必须new KeyboardEvent(keyCode).call(); new UpdateEvent().call(); 才能使用

# Module

注意

必须和主类在同级包下

创建Sprint类.

@ModuleAT
public class Sprint extends Module {
    public Sprint() {
        super("Sprint", "Sprint", Keyboard.KEY_V, Category.MOVEMENT);
    }

    @EventTarget
    private void onUpdate(UpdateEvent updateEvent) {
        Minecraft.getMinecraft().thePlayer.setSprinting(true);
    }
}

提示

加@ModuleAT注解CF4M会自动为您添加到ModuleManager
